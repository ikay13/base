<objective>
## Goal
Move BASE framework files (tasks, context, frameworks, templates) from workspace-local `~/.claude/skills/base/` to global `~/.claude/base-framework/`. The skill entry point (`base.md`) stays at `~/.claude/skills/base/base.md` but all `@` references point to the new global location. Command wrappers updated to match.

## Purpose
Decision global-005 confirmed: framework is user-level, data is workspace-level. Moving framework to global means a user with multiple workspaces gets BASE everywhere without duplicating files. Mirrors PAUL's pattern (global framework + local .paul/ data).

## Output
- `~/.claude/base-framework/` — full framework tree (tasks/, context/, frameworks/, templates/)
- `~/.claude/skills/base/base.md` — references updated to `~/.claude/base-framework/`
- `~/.claude/skills/base/tasks/` etc. — removed (files now live globally)
- `~/.claude/commands/base/*.md` — all 11 reference paths updated
- `apps/base/src/framework/` — dist copy of framework files
- `apps/base/src/skill/base.md` — entry point updated to match
</objective>
