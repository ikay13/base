<objective>
## Goal
Copy all BASE-related files from their current scattered locations into `apps/base/src/` as the distributable source of truth, following the paul-framework package pattern. No rewriting — pure copy.

## Purpose
BASE currently lives across 4 locations (.base/, .claude/skills/base/, .claude/hooks/, ~/.claude/commands/base/). This makes it impossible to push to GitHub or share. The app repo becomes the single source, and the workspace installation becomes a consumer.

## Output
- `apps/base/src/` directory containing all BASE source files organized by type
- `apps/base/package.json` for npm distribution
- `apps/base/bin/install.js` installer (based on PAUL's pattern)
- Complete file manifest — every BASE file accounted for
</objective>
